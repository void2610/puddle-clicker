# 水たまりクリッカー - アイデアノート v2

練習用のシンプルなクリッカーゲーム企画。
**テーマ: 小さな水たまりのまま**

---

## コンセプト

**「小さな水たまりに何かを落として波紋を楽しむ」**

- 水たまりは最初から最後まで小さいまま
- 落とすものが変わっていく
- 波紋のバリエーションが増える

---

## 基本システム

### リソース
**「しずく」** - 唯一の通貨

### クリック
- 水たまりをクリック → 何かが落ちる → 波紋が広がる → しずく獲得

---

## アップグレード

### 1. 落とすもの（クリック効率）

水たまりに落とすものが変わり、波紋の大きさ・獲得しずくが増える。

| 名前 | 効果 | 波紋の演出 |
|-----|------|-----------|
| 指先 | +1しずく（初期） | 小さな波紋 |
| 小石 | +3しずく | やや大きい波紋、ぽちゃん |
| どんぐり | +8しずく | 丸い波紋、ころんと落ちる |
| ビー玉 | +20しずく | きれいな円形波紋 |
| スーパーボール | +50しずく | 跳ねて2回波紋 |

### 2. 波紋の美しさ（獲得ボーナス）

波紋の見た目が変わり、全体の獲得効率が上がる。

| 名前 | 効果 | 演出 |
|-----|------|------|
| 透き通った水 | 獲得+10% | 水の透明度UP |
| 光の反射 | 獲得+25% | 波紋がキラキラ光る |
| 虹色の油膜 | 獲得+50% | 波紋が虹色に変化 |

### 3. 小さな仲間（自動生成）

水たまりの周りに現れる生き物や現象が、自動でしずくを生む。

| 名前 | 効果 | 演出 |
|-----|------|------|
| アメンボ | 毎秒+1しずく | 水面を滑って小さな波紋 |
| 落ち葉 | 毎秒+3しずく | ふわっと落ちてくる |
| 木から落ちる雫 | 毎秒+10しずく | 上からぽたっと1滴落ちる |
| 跳ねるカエル | 毎秒+25しずく | 時々ぴょんと飛び込む |

### 4. 水たまりの深さ（クリック連続ボーナス）

素早くクリックするとボーナス。

| 名前 | 効果 |
|-----|------|
| 浅い水たまり | 初期状態 |
| 少し深い | 連続クリックで+10%ボーナス |
| けっこう深い | 連続クリックで+25%ボーナス |

## ビジュアル案

### 画面構成
```
┌─────────────────────────────┐
│  しずく: 1,234              │
│  毎秒: +15                  │
│                             │
│    🍂      （落ち葉）        │
│                             │
│      ∽∽○∽∽               │ ← 小さな水たまり
│       ∽∽∽∽                │   （サイズは変わらない）
│                             │
│   🐸        🦟              │ ← 周りの仲間たち
│ ────────────────────────── │ ← 地面（アスファルト/土）
├─────────────────────────────┤
│ [小石: 30] [アメンボ: 50]    │
│ [透き通った水: 100]          │
└─────────────────────────────┘
```

### 波紋エフェクト
- 落とすものによって波紋の形・大きさが変わる
- 指先: 小さく繊細
- 小石: ぽちゃんと跳ねる
- スーパーボール: 2回跳ねる演出

### 色彩
- 水たまり: 薄い青〜グレー
- 波紋: 白い輪
- 背景: 灰色のアスファルトに小さな水たまり
- アクセント: 落ち葉のオレンジ、カエルの緑

---

## 進行イメージ

### 序盤（〜50しずく）
- 指先でぽちゃぽちゃ
- 「小石」を買って効率UP

### 中盤（50〜500しずく）
- 「アメンボ」が登場、自動収入開始
- 「透き通った水」で波紋がきれいに

### 終盤（500しずく〜）
- 「跳ねるカエル」で賑やかに
- 「スーパーボール」で大きな波紋
- 小さな水たまりの周りが賑やかな世界に
